# Few-Shot Learning for Image Classification

This project was developed as part of a graduate-level Statistical Machine Learning course and explores the effectiveness of Few-Shot Learning techniques for image classification under limited-data conditions.

The study investigates the performance of Few-Shot Learning approaches compared to traditional supervised learning methods, focusing on classification accuracy, error rates, and training data requirements. Extensive experiments were conducted to evaluate how models generalize when only a small number of labeled samples are available.

As part of this research, a scientific paper was prepared to document the methodology, experimental setup, and obtained results. The repository includes both the implementation in Jupyter Notebook and the accompanying research paper.

Key Contributions:

* Implementation of Few-Shot Learning models for image classification tasks.
* Comparative analysis between Few-Shot Learning and conventional supervised learning approaches.
* Evaluation of model performance under varying dataset sizes.
* Analysis of accuracy, error rates, and generalization capabilities.
* Experimental results and research documentation included.

Technologies:
Python, Jupyter Notebook, Machine Learning, Deep Learning, Computer Vision, Statistical Learning.
